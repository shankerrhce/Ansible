GoVc
====

GoVc is used to manage and automate VMware vSphere 

Find a VM
-----------------

.. code-block:: bash

	# govc find vm -name *jadsh*
	
Delete a VM
---------------
	
.. code-block:: bash

	# govc vm.destroy -name *jadsh*
	
Clone  a VM
---------------
	
.. code-block:: bash

	# govc vm.destroy -name *jadsh*
